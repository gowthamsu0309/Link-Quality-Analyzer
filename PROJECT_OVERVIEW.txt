========================================================
         PROJECT OVERVIEW — Link Quality CSV Dashboard
========================================================

PROJECT NAME   : Link Quality CSV Dashboard
FILE NAME      : CsvDashboardMaven.java
PACKAGE        : dashboard
MAIN CLASS     : CsvDashboard
BUILD TOOL     : Apache Maven
UI FRAMEWORK   : JavaFX
LANGUAGE       : Java 11+

--------------------------------------------------------
WHAT IS THIS PROJECT?
--------------------------------------------------------
The Link Quality CSV Dashboard is a desktop GUI application
built using JavaFX. It is designed to load, parse, and
visualize CSV output files generated by a link quality
analysis pipeline (referred to as "LinkAnalyzerProject").

The application allows a user to:
  - Open CSV files using a file chooser dialog
  - Automatically detect and parse the file type
  - Render meaningful charts and visual summaries
  - Navigate results through a tabbed interface

--------------------------------------------------------
WHY WAS IT BUILT?
--------------------------------------------------------
Data analysis pipelines often produce raw CSV outputs that
are hard to interpret without visualization. This dashboard
bridges the gap between raw data files and human-readable
insights by providing instant, interactive chart views of:

  1. Cluster results       → How data groups are distributed
  2. Change point results  → Where signal behavior changes
  3. AR model results      → How well the model fits and
                             what the coefficients look like

--------------------------------------------------------
WHO IS IT FOR?
--------------------------------------------------------
  - Data analysts working with time-series or link data
  - Researchers using the LinkAnalyzerProject pipeline
  - Developers who need a quick visualization layer for
    CSV data without writing separate plotting scripts

--------------------------------------------------------
WHAT PROBLEMS DOES IT SOLVE?
--------------------------------------------------------
  ✔ No need to import CSVs into Excel or Python manually
  ✔ Instant chart generation from file open
  ✔ Visual coefficient heatmap for quick AR model review
  ✔ Tooltip-powered exploration of exact values
  ✔ Tabbed views prevent chart clutter

--------------------------------------------------------
SUMMARY
--------------------------------------------------------
A lightweight, self-contained JavaFX dashboard that reads
three types of CSV analysis files and renders them as
interactive Pie Charts, Bar Charts, Histograms, and a
color-coded Coefficient Heatmap — all within a clean
tabbed desktop UI, built and run with Apache Maven.
