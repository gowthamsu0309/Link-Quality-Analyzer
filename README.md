## рҹ“қ Description

The Link Quality CSV Dashboard is a desktop application built using Java and JavaFX, packaged and managed with Apache Maven. It is designed to help data analysts and researchers instantly visualize the output CSV files generated by a link quality analysis pipeline, without needing external tools like Excel or Python scripts.

The application launches a clean, resizable desktop window with a top menu bar and a tabbed chart panel. Users can load three types of CSV files directly through the File menu using a native OS file chooser dialog.

For **Cluster CSV** files, the dashboard parses each row's cluster label and renders both a Pie Chart showing the proportional distribution of clusters and a Bar Chart displaying the raw count of series per cluster вҖ” giving analysts an instant picture of how data has been grouped.

For **Change Point CSV** files, the dashboard counts the number of detected change points per series and plots them as a Bar Chart sorted by series index. This helps identify which series have the most signal variability at a glance.

For **AR Model CSV** files, the dashboard produces two visualizations. The first is a 10-bin histogram of residual variance values, showing how well the model fits across all series. The second is a color-coded Coefficient Heatmap rendered as a scrollable grid, where each cell represents one AR coefficient. Cells are colored on a BlueвҶ’WhiteвҶ’Red scale вҖ” blue for strong negative values, white near zero, and red for strong positive values. Hovering over any cell shows its exact value via a tooltip.

The application is lightweight, self-contained, and requires only Java 11 and Maven to build and run, making it easy to set up and share across teams.
